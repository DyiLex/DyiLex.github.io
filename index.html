Yarn初安装使用，竟出现这种事？！
发表于2024-12-21
咋了，细说？今天新开一demo，安装使用Yarn，遂出现： 怎么个事？于是打开浏览器就是一套操作，很好，大家都碰见过这个问题，那我就放心啦！原因竟是： Yarn的执行文件路径与全局安装路径不同分区！ 然后又是一顿操作，嗨呀还真是！ 就这么办！导致这种情况的原因竟然是分区不同，不明觉厉哦Yarn！知道原因就好办了，直接把Yarn的全局安装分区改到执行文件分区就行了，Just do it！ 12yarn config set global-folder "D:\Softwares\..\data\yarn_global"yarn config set cache-folder "D:\Softwares\..\data\yarn_cache" 执行上面两条命令，把Yarn的全局安装路径和缓存路径改到D分区，然后检查一下， 不忘初心，又回到最初的起点，执行！
